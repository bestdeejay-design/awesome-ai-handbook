## Быстрый старт
```bash
# Установка Ollama (простейший способ начать)
brew install ollama    # macOS
# curl -fsSL https://ollama.com/install.sh | sh  # Linux

# Запуск модели
ollama run qwen3:8b

# Готово! Спросите что-нибудь в появившемся чате.
```
