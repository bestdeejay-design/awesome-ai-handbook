## Перед началом

Убедитесь, что:
1. Установлена Ollama — [local-models/getting-started.ru.md](../../local-models/getting-started.ru.md)
2. Запущена модель `qwen3.5:4b` — `ollama pull qwen3.5:4b`
3. Установлен Python 3.10+ с `pip install requests`
