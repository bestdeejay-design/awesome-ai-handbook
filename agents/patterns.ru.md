# 🧩 Паттерны AI-агентов

Архитектурные паттерны для построения агентов поверх LLM.

[← AI-агенты](README.ru.md)

---

## Что такое AI-агент

AI-агент = LLM + память + инструменты + цикл принятия решений.

В отличие от обычного чата, агент может:
- Планировать действия
- Использовать инструменты (поиск, код, API)
- Помнить контекст между шагами
- Анализировать и улучшать свои результаты

---

## Паттерны

| Паттерн | Описание | Когда использовать |
|---|---|---|
| **ReAct** | Reasoning + Acting по очереди | Простые задачи с инструментами |
| **Plan-and-Execute** | Сначала план, потом выполнение | Сложные многошаговые задачи |
| **Multi-Agent** | Несколько агентов с разными ролями | Большие задачи, разделение ответственности |
| **Reflection** | Агент проверяет и улучшает свой вывод | Когда важно качество |
| **Tool Use** | Агент выбирает и вызывает инструменты | Когда нужны внешние данные/действия |

---

## ReAct (Reason + Act)

```
Thought → Action → Observation → Thought → ...
```

Модель чередует рассуждение и вызов инструментов. Простой и понятный паттерн для первого агента.

**Подходит для:** поиск в интернете, калькулятор, чтение файлов.

---

## Plan-and-Execute

```
Plan (шаги 1..N) → Execute step 1 → Execute step 2 → ...
```

Сначала строится полный план, затем шаги выполняются последовательно или параллельно.

**Подходит для:** миграции кода, многофайловые рефакторинги, исследовательские задачи.

---

## Multi-Agent

Несколько агентов с ролями (писатель, рецензент, исполнитель) обмениваются сообщениями.

**Подходит для:** генерация контента с проверкой, code review, исследовательские пайплайны.

**Фреймворки:** [CrewAI](frameworks.md#crewai), [AutoGen](frameworks.md#autogen-microsoft).

---

## Reflection

Агент генерирует ответ, затем критикует его и улучшает в следующей итерации.

**Подходит для:** написание текстов, сложные рассуждения, когда нужна самопроверка.

---

## Tool Use

Модель получает описание инструментов (JSON Schema) и решает, какой вызвать.

**Локально:** Ollama поддерживает tool calling и structured output — см. [ollama-for-agents.md](ollama-for-agents.ru.md).
