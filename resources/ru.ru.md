## Как добавить ресурс

1. Убедитесь, что ресурс активен (публикации за последние 3–6 месяцев)
2. Добавьте строку в соответствующий подраздел с описанием «зачем это полезно»
3. Укажите `last-verified: YYYY-MM-DD`
4. Откройте PR

Формат записи:

```markdown
- **[Название](URL)** — краткое описание. `last-verified: 2026-07-24`
```
