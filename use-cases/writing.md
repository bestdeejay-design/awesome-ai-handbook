## Рекомендуемые модели

| Задача | Модель | Запуск | Почему |
|--------|--------|--------|--------|
| **Быстрые черновики** | Qwen 3.5 4B | `ollama run qwen3.5:4b` | 28–35 tok/s, хороший русский |
| **Качественные тексты** | Qwen 3.5 9B | `ollama run qwen3.5:9b` | Лучший баланс на 16 GB |
| **Классика** | Llama 3.1 8B | `ollama run llama3.1:8b` | Стабильный стиль |
| **Аналитика / отчёты** | Q5_K_M вместо Q4 | см. [quantization.md](../local-models/quantization.md) | Чуть выше качество формулировок |

---

## Квантизация для текстов

Для письма и аналитики имеет смысл **Q5_K_M** вместо Q4_K_M — если модель влезает в RAM:

| Формат | Когда |
|--------|-------|
| Q4_K_M | По умолчанию, черновики |
| Q5_K_M | Финальные тексты, аналитика |
| Q6_K | Юридические / медицинские формулировки |
